# Peta Jalan Tol di Jawa Barat
Menampilkan peta Jawa Barat dan jalan tolnya

## Library Used
```
pandas
matplotlib.pyplot
geopandas
shapely.geometry
```

### Install Library:
```
pip install pandas
pip install matplotlib
pip install geopandas
pip install shapely
```

### Buat windows - Kalau gagal install geopandas:
```
pip install pipwin
pipwin install gdal
pipwin install fiona
pip install geopandas
```
